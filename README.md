# Crowdfunding on Solana

A crowdfunding platform built on the Solana blockchain, allowing creators to initiate fundraising campaigns and receive donations. The platform supports various essential features, including creating campaigns, making donations, withdrawing funds, and contract management for the administrator.

## Features

* **Campaign Creation** : Creators can initiate campaigns with a title, description, goal, and deadline.
* **Donations** : Users can donate to campaigns as long as the campaign is active.
* **Withdrawal** : Campaign creators can withdraw funds after reaching the goal and upon campaign completion.
* **Contract Pause** : The administrator can pause and resume the contract as needed.

## Prerequisites

* **Rust** : Make sure you have [Rust installed]().
* **Anchor** : Install Anchor for Solana smart contract development by following the [official guide]().
* **Solana CLI** : Install the [Solana CLI]().
* **Node.js and npm** : Required for running client scripts. Download them from [Node.js official site](https://nodejs.org/).

## Project Structure

* **programs/crowdfunding** : Contains the main Rust program.
* **scripts** : Contains TypeScript scripts for interacting with the deployed program.
* **migrations** : Deployment scripts for Anchor.

## Contract Functions

### Rust Functions (in `programs/crowdfunding/src/lib.rs`)

* **initialize** : Sets up the admin account with the authority key.
* **create_campaign** : Creates a campaign with specified title, description, goal, and deadline.
* **donate** : Allows users to donate to a specific campaign.
* **withdraw** : Allows campaign creators to withdraw funds after the campaign has ended successfully.
* **toggle_pause** : Pauses or resumes the contract.

## Error Codes

The program has custom error codes defined in `ErrorCode` for managing expected failures, such as:

* `CampaignEnded`: Attempted donation to an ended campaign.
* `CampaignNotEnded`: Attempted withdrawal before the campaign has ended.
* `GoalNotReached`: Withdrawal attempted before reaching the funding goal.
* `Unauthorized`: Unauthorized action.
* `ContractPaused`: Action attempted while the contract is paused.
